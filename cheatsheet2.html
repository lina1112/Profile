git add Adds all new files, known as staging

How to declare + assign variables var declares a variable then you choose what to assign it to. E.g. var name string = "leah"
Short assignment (:=) shorthand for declare and assign. (=) to assign
if/else statements This is used to carry out code if a condition is true or false.
E.g. if name == "leah" {
// do this code if condition is true
} else {
// do this code if condition is false
}
For Loops The for loop is used when you want to repeat the same code a specified number of times. It can take 3 statements
E.g. package main
import ("fmt")

func main() {
for i:=0; i < 5; i++ {
fmt.Println(i)
}
}
The first statement(i=0) sets the value at 0 .
The second Evaluates each loop, if its true it continues if not the loop ends.
The last statement (i++)increases the loop counter value by 1 each time.

add your code that defines what the function should do, inside curly braces {}
